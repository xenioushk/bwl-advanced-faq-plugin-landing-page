What You Need to Know

Frequently Asked Questions And Answers

What are the technical requirements of the FAQ plugin?

You need at least WordPress version 4.8+ installed for this plugin to work properly. It is strongly recommended that you always use the latest stable version of WordPress to ensure all known bugs and security issues are fixed. Other requirements:

  • PHP version 5.3 or greater.
  • MySQL version 5.0.15 or greater.

Does the search box work in a mobile device?

Yes, plugin search box support and work perfectly in mobile devices. FAQ Search box has a clear ( X ) button that helps users to clear the search field text.

Does plugin have the FAQ row toggle option?

Yes, FAQ plugin comes with FAQ row toggle feature. Please check the following steps to activate it.

  1. Go to FAQ Settings >> Reading Settings section.
  2. From the Collapsible Accordion: drop down, select Show All FAQ Answer Closed option.
  3. Finally, click on the Save button.

Can I use the FAQ plugin with WooCommerce?

Yes, FAQ plugin is fully compatible with WooCommerce latest version. Using the shortcode, you can insert the FAQ into the product details.

However, If you need to create a custom tab for FAQ, then install and activate FAQ Tab For WooCommerce addon. This addon allows you to use the existing FAQ content as WooCommerce product FAQ. So, no need to create the same FAQ repeatedly for the WooCommerce product. That is super easy to use and very efficient. Please note that, you will get the addon with BWL Advanced FAQ Manager zip file.
